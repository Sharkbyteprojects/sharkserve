# How to:
### Install
- Make an file and name it `sharkserve.js` and write in it `require("sharkserve");`
- run `npm init -y`
- run `npm i sharkserve`

### Use
- Serve dir /home/ from port 8081:
- `node sharkserve --port=8081 --dir=/home`
- Standard settings: dir . port 8080
- `node sharkserve`
